### Run project 
```sh
- git clone https://github.com/Shokr/CloudInn-Task.git
- cd CloudInn-Task
- virtualenv -p python3.8 venv
- source venv/bin/activate
- pip install -r requirements.txt
- python main.py
```

### Usage
- Enter Unit Name:-  
    - Type Age of Empires II Unit Name like Crossbowman, Skirmisher, etc...
